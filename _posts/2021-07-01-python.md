# 1. *args
: 정해지지 않은 매개변수 입력받기

```python
def avg_numbers(*args):
    result = 0
    for i in args:
        result += i
    return result / len(args)
    
print(avg_numbers(1,2,3,4,5))
```

: 파일을 `자동으로 여닫아줌`

with open('test.txt', 'w') as f:

```

# 4. argv

```python
import sys
# 파일 이름이 [0]이기 때문에 [1]부터
args = sys.argv[1:]
a = int(args[0])
c = int(args[2])

if args[1] == '+':
    print(a + c)
elif args[1] == '-':
    print(a - c)
elif args[1] == '*':
    print(a * c)
elif args[1] == '/':
    print(a / c)
else:
    print("없는 연산자 입니다.")
``
